linux包管理器

发表于2024-11-14|更新于2024-11-14
|字数总计:350|阅读时长:1分钟|阅读量:

引子

游走在各个发行版，每个包管理器都得用一用，记录一下。没有多么高端的使用方法，都是一些简单的命令，后续可能会添加吧。 😋

apt

查看

搜索可用的包apt search keyword

查看已经安装的包apt list --installed

更新`

更新包列表apt update

安装可用更新apt upgrade

安装

apt install package

卸载

删除软件包apt remove package

删除软件包及其相关依赖apt autoremove package

删除软件包和配置文件apt purge package

dpkg

查看

查看已安装的包dpkg -l

查看包相关的文件dpkg -L package

安装

dpkg -i xx.deb

卸载

dpkg -r package

yum

查看

搜索可用的包yum search keyword

查看已经安装的包yum list installed

更新

更新元数据yum makecache

更新可用软件包yum update

安装

yum install package

卸载

删除软件包yum remove package

删除软件包和其相关依赖yum autoremove package

rpm

查看

查看已安装的包rpm -qa

安装

rpm -i package

卸载

rpm -e package

pacman

查看

搜索软件包pacman -Ss keyword

查看已安装的包pacman -Q package

更新

同步更新列表pacman -Sy

更新系统pacman -Syu

强制同步更新列表并更新系统pacman -Syyu

安装

pacman -S package

卸载

删除软件包pacman -R package

删除软件包及其相关配置pacman -Rns package
